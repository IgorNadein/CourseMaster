# Changelog: 2025-12-28 - Фаза 4.2 и 4.8

## Фаза 4.2: Миграция данных в Step-формат

### Создан management command `migrate_to_steps`
- Команда для конвертации существующих уроков в Step-формат
- Расположение: `courses/management/commands/migrate_to_steps.py`

### Поддерживаемые типы конвертации
| Старый формат | Новый формат Step |
|---------------|-------------------|
| Lesson (video) | Step (type=video) |
| Lesson (article) | Step (type=text) |
| Quiz с вопросами | Step[] (type=quiz_single/quiz_multiple) |
| Assignment | Step (type=free_answer) |

### Использование команды
```bash
# Предпросмотр плана миграции
python manage.py migrate_to_steps

# Выполнить миграцию
python manage.py migrate_to_steps --execute

# Удалить все шаги и мигрировать заново
python manage.py migrate_to_steps --reset --execute
```

### Результат миграции
- 63 урока → 63 шага
- 40 видео → 40 Step(video)
- 3 статьи → 3 Step(text)
- 10 тестов → 10 Step(quiz_single/quiz_multiple)
- 10 заданий → 10 Step(free_answer)
